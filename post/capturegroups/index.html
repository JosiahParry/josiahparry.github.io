Capture Groups

My new favorite regular expression trick

Jan 1, 0001 3 min read

capture groups referencing those capture group

Working with geniusR has made me confront my fear of regular expressions. In a recent pull request I almost entirely rewrote the internals of the main function in the package. The biggest difficulty I faced was being able to insert new lines by detected string paterns. I stumbled across some syntax that I’ve never seen before which solved the problem. The syntax involves capture groups and back references.

Capture groups are denoted by placing a regular expression (regex) between parentheses (i.e. ([regex])). Capture groups are then given a numbered reference group (i.e \1).

Let’s hop into a few examples to see how it works. I’m going to use stringr for all of string manipulations. For a first example we’ll just look at how we can extract a word from a string. For this example we’ll use the function str_extract(). Like all good tidyverse functions the first argument is the data, or in the case the string that we will be manipulating. The second argument is the pattern that we are looking for. For this example we’ll use the string "Merry Christmas", and we’ll extract the word "Christmas".

library(stringr)

# create a string to extract from 
my_string <- "Merry Christmas"

# extract the word christmas.
str_extract(my_string, "Christmas")
## [1] "Christmas"

Staying in the Christmas spirit we will take this example and expand it on the first verse of Mariah Carey’s famous “All I want for Christmas Is You”.

all_i_want <- "I-I-I don't want a lot for Christmas
There is just one thing I need
I don't care about the presents
Underneath the Christmas tree
I just want you for my own
More than you could ever know
Make my wish come true
All I want for Christmas is you, yeah"

There is a version of str_extract() which will let us take every instance of our detected pattern, unsuprisingly this is str_extract_all(). It works identically.

str_extract_all(all_i_want, "Christmas")
## [[1]]
## [1] "Christmas" "Christmas" "Christmas"

Now we could replace this pattern with something else by using the str_replace_all() function. The only difference we need to do here is provide a replacement string. Here we’ll replace "Christmas" with "CHRISTMAS".

str_replace_all(all_i_want, "Christmas", "CHRISTMAS")
## [1] "I-I-I don't want a lot for CHRISTMAS\nThere is just one thing I need\nI don't care about the presents\nUnderneath the CHRISTMAS tree\nI just want you for my own\nMore than you could ever know\nMake my wish come true\nAll I want for CHRISTMAS is you, yeah"

We can turn our search pattern into a capture group. If wew

str_replace_all(all_i_want, "(Christmas)", "\\1")
